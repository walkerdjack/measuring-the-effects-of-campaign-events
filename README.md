### Replication Materials for Lockhart, M., Huber, G. A., Gerber, A. S., & Walker, J. D. II. [Date]. "Measuring the Effects of Campaign Events: Specifying and Comparing Estimates of the Effect of Trump's Conviction." Political Science Research and Methods.

This replication package contains all data and code necessary to reproduce the figures and tables in the main text and appendix of the manuscript. All data are anonymized.

Corresponding author: Gregory A. Huber, [gregory.huber\@yale.edu](mailto:gregory.huber@yale.edu).

#### Data file:
1. `psrm_lockhart_etal_public.dta`

#### Code files:
1. `analysis.do` creates all tables in the main text and appendix in Stata.
2. `analysis.R` creates the figure in the appendix in R.

IMPORTANT: The data file must be located in the same folder as the code files.
